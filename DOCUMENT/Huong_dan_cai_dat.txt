HƯỚNG DẪN CÀI ĐẶT VÀ CHẠY DỰ ÁN VAYCUOIGIS

BƯỚC 1: TẠO CƠ SỞ DỮ LIỆU
1. Mở pgAdmin, tạo một Database mới tên là: vaycuoigis
2. Click chuột phải vào database vừa tạo, chọn Restore (hoặc dùng Query Tool) để nạp file "vaycuoigis_backup.backup" vào hệ thống.

BƯỚC 2: CẤU HÌNH DATABASE
Mở file vaycuoigis/settings.py, tìm đến mục DATABASES và chỉnh sửa lại 'USER' và 'PASSWORD' cho khớp với PostgreSQL trên máy tính sử dụng.

BƯỚC 3: CÀI ĐẶT MÔI TRƯỜNG 
1. Mở Terminal tại thư mục dự án.
2. Tạo môi trường ảo: 
   - Trên Windows: python -m venv venv
   - Trên macOS/Linux: python3 -m venv venv
3. Kích hoạt môi trường:
   - Trên Windows: venv\Scripts\activate
   - Trên macOS/Linux: source venv/bin/activate
4. Cài đặt thư viện từ file requirements.txt: pip install -r requirements.txt
!!!Trường hợp nếu bị lỗi thì chạy từng câu lệnh pip:
   - pip install django-jazzmin
   - pip install openpyxl
   - pip install asgiref
   - pip install Django
   - pip install et_xmlfile
   - pip install pillow
   - pip install psycopg2-binary
   - pip install sqlparse
   - pip install typing_extensions

BƯỚC 4: CHẠY MÁY CHỦ
Gõ lệnh: python manage.py runserver
Truy cập vào link http://127.0.0.1:8000/ để xem giao diện web.
